# Gor
Gor(from "Go R") is a dynamically-typed, R-inspired language

Here's a "hello world" example:
```
? comments are done with '?'

? assign our message to the variable 'hello'
hello <- "Hello, Catdog!"

? print 'hello' to the screen
puts(hello)
```
